# A-B 区段列车运行图编制程序说明

## 1. 概述

本程序基于题目 **4.11** 的要求，实现单线半自动闭塞区段（A—B）的列车运行图编制，包括通过能力计算、运行图铺画和质量指标计算。

**输入**：区段基础数据（站间距、运行时分、列车参数等）  
**输出**：控制台计算报表 + 列车运行图 PNG + 附表1/2 Excel

- **运行方式**：`python train_diagram.py`（运行图）+ `python calc_tables.py`（附表）
- **依赖**：Python 3.x + matplotlib + openpyxl

---

## 4. 核心算法

### 4.1 平行运行图通过能力

对于单线半自动闭塞区段，采用**限制区间法**：

$$T_{周期} = t_{下行}' + t_{上行}' + \tau_{不} + \tau_{会}$$

$$t' = t_{纯运行} + t_{起} + t_{停}$$

$$N_{平行} = \left\lfloor \frac{1440 - T_{天窗}}{T_{周期}^{限制}} \right\rfloor$$

**流程**：
1. 遍历 6 个区间，计算各区间的 $T_{周期}$
2. 取 $T_{周期}$ 最大的区间为**限制区间**
3. 用限制区间的周期计算通过能力

**结果**：限制区间 d—e，$T_{周期} = 61$ min，$N_{平行} = 22$ 对/天。

### 4.2 非平行运行图通过能力

在平行运行图基础上，扣除不同类型列车的影响：

$$N_{非} = N_{平行} - \left[ \varepsilon_{客} \cdot n_{客} + (\varepsilon_{快货} - 1) \cdot n_{快货} + (\varepsilon_{摘挂} - 1) \cdot n_{摘挂} \right]$$

其中 $n$ 均为**对数**（非列数）：

| 扣除项 | 计算 | 扣除值 |
|--------|------|--------|
| 旅客列车 | 1.3 × 2 | 2.6 |
| 摘挂追加 | (1.6-1) × 1 | 0.6 |
| 快货追加 | (1.6-1) × 3 | 1.8 |
| **总扣除** | | **5.0 对** |

**结果**：$N_{非平行} = 22 - 5.0 = 17$ 对/天。

### 4.3 列车运行图铺画

采用**时间—距离坐标系**：
- 横轴：时间（0:00—24:00），每 10 分钟一格
- 纵轴：车站位置，按重车方向运行时间比例缩放

**单线冲突检测机制**：
- 每个区间同一时刻只能被一列车占用
- 区间占用记录含 τ不/τ会缓冲：有效占用 = [进入-τ会, 离开+τ不]
- 后续列车检测到冲突时，在上一站等待至区间清空
- 调度顺序：旅客列车 → 摘挂列车 → 下行/上行交替

**铺画逻辑**：

```
对每列车：
  1. 确定始发时刻（旅客列车固定，货物列车均匀分布）
  2. 沿方向逐站计算，每进入一个区间前检测冲突
  3. 如有冲突，在上一站等待，回写更新出发时间
  4. 收集各站到发点，按时间排序后连线
```

**绘图函数** `draw_train_diagram()`：
- 收集每列车各站到发时刻 → 按时间排序 → matplotlib 连线
- 不同列车类型使用不同颜色：红(旅客)、蓝(直达)、绿(区段)、紫(摘挂)
- 标注天窗区域（灰色）

### 4.4 会车方案

根据两端车站股道数选择 4 种标准会车方案：

| 区间 | 左站股道 | 右站股道 | 会车方案 |
|------|----------|----------|----------|
| A-a | A(5) | a(3) | (a) 两端不停车通过 |
| a-b | a(3) | b(2) | (c) 下行不停车通过 |
| b-c | b(2) | c(3) | (d) 上行不停车通过 |
| c-d | c(3) | d(3) | (a) 两端不停车通过 |
| d-e | d(3) | e(2) | (c) 下行不停车通过 |
| e-B | e(2) | B(5) | (d) 上行不停车通过 |

### 4.5 质量指标计算

#### (a) 速度指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 技术速度 | $v_{技} = \frac{\sum L}{\sum t_{纯运行}}$ | 不含停站的纯运行速度 |
| 旅行速度 | $v_{旅} = \frac{\sum L}{\sum t_{旅行}}$ | 含停站的实际旅行速度 |
| 速度系数 | $\beta = \frac{v_{旅}}{v_{技}}$ | 反映停站影响程度（< 1） |

#### (b) 机车运用指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 机车全周转时间 | $T_{周} = 2 \cdot t_{单程} + t_{折返}^A + t_{折返}^B$ | 一次完整循环 |
| 机车日车公里 | $S_{日} = \frac{1440}{T_{周}} \cdot 2L \cdot n_{机车}$ | 总走行公里/日 |
